## TECHNICAL QUESTIONS - QUICK ANSWERS

**Q: What tech stack?**
A: React.js frontend, Flask backend, localStorage for data, ML with scikit-learn

**Q: How do you predict periods?**
A: Average cycle length calculation + ML model for pattern recognition

**Q: Voice recognition accuracy?**
A: Using Web Speech API + custom NLP for symptom extraction, ~85% accuracy

**Q: PCOD prediction model?**
A: Rule-based assessment + training ML model with medical datasets

**Q: Scalability?**
A: Serverless architecture on Vercel, can scale to 100K+ users

**Q: Data security?**
A: Client-side encryption, JWT auth, HTTPS only, no third-party tracking

**Q: Mobile app?**
A: React Native version in development, launching in 3 months

**Q: Monetization?**
A: Freemium model - basic free, premium features (telemedicine, advanced insights) paid
